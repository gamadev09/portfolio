# Proyecto de Competencias Digitales

Este proyecto es un portafolio digital desarrollado para la materia de Competencias Digitales.

## Información del proyecto
- Universidad: Universidad Nacional de Ingeniería
- Grupo: 8
- Objetivo: presentar de forma organizada y visual los trabajos, recursos y contenidos relacionados con el curso.

## Contenido
El sitio incluye secciones para:
- Presentación del equipo
- Definición del tema
- Video relacionado
- Informe
- Infografía
- Galería de imágenes

## Cómo visualizar el proyecto
1. Abre el archivo `index.html` en tu navegador.
2. Asegúrate de que los archivos `styles.css` y `script.js` estén en la misma carpeta.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- Tailwind CSS
